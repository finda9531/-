## 本ツールの性質および法的・技術的スタンスについて

本ツールは、ユーザーの利便性向上を目的とした **“入力補助ツール”** であり、いわゆる **“自動ログインツール”** とは一線を画すものです。以下の通り、技術的および法的な観点からその正当性を定義します。

### 1. 認証プロセスの非関与（技術的独立性）
本ツールは、大学側のサーバーと直接通信を行ったり、認証プロトコルをバイパス（回避）したりする機能は一切有していません。
* **動作対象:** ユーザーのブラウザ上の “入力フォーム（DOM）” に限定されています。
* **法的解釈:** サーバーに対して “偽りの命令” を送るものではないため、不正アクセス行為禁止法における “識別符号の入力” を不正に行うツールには該当しません。

### 2. ユーザーの操作主体性（意思の介在）
本ツールの動作は、常にユーザーの明示的な操作を起点としています。
* **動作原理:** あらかじめユーザー自身が設定・保存した “カード（ログイン情報）” を、ユーザーの実行指示に基づいて適切なフィールドへ転記する “デジタルな代筆” に過ぎません。
* **比較:** これはブラウザ標準の “オートフィル機能” や “パスワードマネージャー” と同等の仕組みであり、操作の主体は常にユーザーにあります。

### 3. 刑事罰の構成要件に関する見解
本ツールは、大学のシステムの正常な動作を妨げたり、過度な負荷をかけたりするものではありません。
* **偽計業務妨害罪:** 手入力の代替として機能するのみであり、システム側に「人間ではないものが不正に操作している」と誤認させるような隠蔽工作も行わないため、業務妨害には該当しないと解釈しています。
* **電子計算機使用詐欺罪:** 刑法における構成要件である “虚偽の情報や不正な指令” の与えには該当しません。
